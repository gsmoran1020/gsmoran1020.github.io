Django Blog

A fully functional blog application that allows users to engage with a variety of common blogging features. Features such as creating, reading, updating, and deleting posts are present. Along with basic CRUD features, users can like and add comments to other users' posts. On the main page users are able to search for articles if they are looking for specific posts. This applicaiton is built entirely on Django and was originally a tutorial project that I modified and made additions to.
